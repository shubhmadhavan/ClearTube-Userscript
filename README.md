# ClearTube-Userscript
A userscript to customise Youtube homepage, recommendations, comments, shorts tab etc 

## How to Use
- Download a compatible userscript manager extension for your browser
- Popular options are:
  - [Tampermonkey](https://www.tampermonkey.net/)
  - [Violentmonkey](https://violentmonkey.github.io/get-it/)
- Then simply add the ClearTube-1.3.user.js file from this repository
- On a webpage, click on your userscript manager extension
  - For Tampermonkey, it should look something like this: 
 <img width="230" alt="2025-11-02 (09-55-37)" src="https://github.com/user-attachments/assets/5f9cbf30-ffa5-47bc-8956-b68951418a73" />

- Then simply toggle features as per your need
